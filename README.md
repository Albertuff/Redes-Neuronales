# Redes-Neuronales

Las bases de datos se descargaron de la pagina Kaggle:
  * https://www.kaggle.com/datasets/biaiscience/dogs-vs-cats
  * https://www.kaggle.com/datasets/andreazzini/international-airline-passengers
